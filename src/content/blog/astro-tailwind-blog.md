## 为什么选择 Astro？

在构建这个博客时，我选择了 Astro 作为静态网站生成器，主要原因如下：

### 1. 极致性能

Astro 采用了一种独特的方法来构建网站：默认情况下，它会将所有的 JavaScript 发送到客户端。这意味着：

- 更快的页面加载速度
- 更好的 SEO
- 更低的带宽消耗

### 2. 开发体验

Astro 提供了优秀的开发体验：

- 支持 TypeScript
- 热模块替换 (HMR)
- 直观的组件语法
- 丰富的集成生态

### 3. 内容管理

Astro 内置了对 Markdown 和 MDX 的支持，使得内容管理变得非常简单。

## Tailwind CSS 的优势

配合 Tailwind CSS，可以快速构建出现代化的 UI：

### 1. 快速开发

使用 Tailwind 的实用类，可以快速构建样式，无需频繁切换文件。

### 2. 一致性

Tailwind 提供了一致的设计系统，确保整个网站的样式保持统一。

### 3. 响应式设计

Tailwind 的响应式修饰符使得创建响应式布局变得非常简单。

## Apple 风格设计

这个博客采用了 Apple 风格的设计理念：

- **大圆角**：使用较大的圆角值（20px+）
- **柔和阴影**：使用微妙而优雅的阴影效果
- **毛玻璃效果**：导航栏使用背景模糊效果
- **流畅动画**：使用 ease-out 缓动函数
- **大量留白**：给内容足够的空间呼吸

## 总结

Astro + Tailwind CSS 是一个强大的组合，适合构建高性能的静态网站。如果你也想创建一个个人博客，我强烈推荐尝试这个技术栈。
